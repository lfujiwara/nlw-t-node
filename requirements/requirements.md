# General requirements

## Add tags (types of valid compliments)

- Admin only
- Unique & required name

## Add compliments

- Sender ID
- Receiver ID
- Tag ID
- DateTime of event
- Prevent self compliment
- Verify if both sender and receiver exists
- Require auth

## AuthN/AuthZ

- JWT
- RBAC for routes

## Queries

- List users
- List tags
- List compliments per user

## User sign up

- Unique & required email
